## 如何切换字体？

### 方法一：修改配置文件（推荐）

1. 打开文件：`src/app/layout.tsx`
2. 找到第60行：
```typescript
// 当前使用的字体（改这里切换字体）
const currentBrandFont = zCoolKuaiLe; // 👈 站酷快乐体 - 活泼欢快
```
3. 将 `zCoolKuaiLe` 改为你想要的字体变量名
4. 保存文件，浏览器会自动刷新

### 方法二：切换CSS变量

1. 打开文件：`tailwind.config.ts`
2. 找到第13行：
```typescript
brand: ["var(--font-style-3)", "system-ui", "sans-serif"], // 当前使用字体
```
3. 将 `--font-style-3` 改为对应的编号：
   - `--font-style-1` = 站酷小薇
   - `--font-style-2` = 站酷庆科黄油体
   - `--font-style-3` = 站酷快乐体（当前）
   - `--font-style-4` = 刘建毛草
   - `--font-style-5` = 志莽星
